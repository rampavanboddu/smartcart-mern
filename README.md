# SmartCart – MERN E-Commerce Application

## About Project

This project is a full-stack web-based e-commerce application developed using the MERN stack. It allows users to browse products, manage their cart, and place orders. The system also includes authentication and backend API integration for handling data securely.

---

## Technologies Used

* React.js (Frontend)
* Node.js
* Express.js
* MongoDB (Database)
* HTML, CSS

---

## Features

### User Features

* User Registration and Login
* Browse Products
* Add to Cart
* Place Orders
* View Order Details

### Backend Features

* REST API for products and orders
* JWT Authentication
* MongoDB Database Integration

---

## Database

Database: MongoDB Atlas

Collections used:

* users
* products
* orders

---

## How to Run

### Backend Setup

* Open terminal
* Navigate to server folder:

```bash
cd Server
```

* Install dependencies:

```bash
npm install
```

* Create `.env` file and add:

```bash
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
```

* Run server:

```bash
node server.js
```

---

### Frontend Setup

* Open new terminal
* Navigate to client folder:

```bash
cd client
```

* Install dependencies:

```bash
npm install
```

* Start React app:

```bash
npm start
```

---

## Run Application

* Backend: http://localhost:5000
* Frontend: http://localhost:3000
